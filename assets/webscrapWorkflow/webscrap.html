Web Scraping and Analysis Pipeline

1. User Input Processing

scrapfast() / search_and_scrape()

→ Process keywords and parameters

2. Search Engine Querying

scrape_website_content()

→ Multiple search engines (Google, Bing, DuckDuckGo, etc.)

3. URL Collection and Validation

BeuatifulSoup.select(engine['result_selector'])

→ Extract and validate URLs from search results

4. Content Scraping

langchain WebBaseLoader.load()

→ Extract content from validated URLs

5. Content Processing

Deepseek.get_content_summary()

→ Summarization and text analysis

6. Sentiment Analysis

perform_overall_sentiment_analysis()

→ Generate sentiment metrics and insights

Function Input Output Description
